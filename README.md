# Archit's Life Dashboard

Personal nutrition, meal prep, grocery & wellness tracker.

🔗 **Live at**: [https://archit1823.github.io/archit-life-dashboard](https://archit1823.github.io/archit-life-dashboard)

## Features
- 📅 Weekly meal plan with daily breakdown
- 🛒 Interactive grocery checklist (Mayuri, Whole Foods, Costco)
- 💊 Supplement schedule with timing
- 🍳 Meal prep guide (Sunday batch + Wednesday refresh)
- 📱 PWA — installable on phone (Add to Home Screen)
- 🌙 Dark theme, mobile-responsive

## Setup
1. Fork/clone this repo
2. Go to **Settings → Pages → Source: Deploy from branch → main**
3. Wait 1-2 minutes → live at `archit1823.github.io/archit-life-dashboard`

## Install on Phone
1. Open the URL in Safari (iOS) or Chrome (Android)
2. Tap **Share → Add to Home Screen**
3. Opens as a full-screen app — no browser bar!

## Updating
Edit `index.html` data sections to update meals, grocery, or supplements.
Push to `main` — GitHub Pages auto-deploys in ~30 seconds.
